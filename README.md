# 🌸 ManhwaRec Assistant: Интеллектуальная система рекомендаций манхв

## Описание проекта
**ManhwaRec Assistant** — это прототип рекомендательной системы, использующей архитектуру **Retrieval-Augmented Generation (RAG)** для подбора корейских вебтунов. 

Система осуществляет гибридный поиск: сочетает векторный поиск по смыслу с фильтрацией по метаданным (год выпуска, статус публикации, рейтинг пользователей). Для генерации финальной рекомендации с обоснованием используется легковесная языковая модель (Qwen2.5-0.5B-Instruct).

## Описание данных

В проекте используется датасет [**Manhwa Industry Evolution**](https://www.kaggle.com/datasets/artheon/manhwa-industry-evolution-2000-2026), содержащий метаданные более чем о 5000 тайтлах.

**Структура данных:**
*   `title`: Название манхвы
*   `description`: Очищенное описание сюжета (используется для векторизации)
*   `genres`: Список жанров через запятую
*   `year`: Год начала публикации
*   `status`: Статус (`FINISHED`, `RELEASING` и др.)
*   `score`: Средний пользовательский рейтинг (0-100)
*   `popularity`: Количество пользователей, читающий манхву на сайте AniList

## Краткое описание методов

В проекте реализован RAG пайплайн, состоящий из следующих этапов:

<img width="872" height="482" alt="image" src="https://github.com/user-attachments/assets/b99b83e1-f3e8-4d68-b88f-ed9353f59061" />


1.  **Векторизация:**
    *   Используется модель `sentence-transformers/all-MiniLM-L6-v2`
    *   Она преобразует текстовые описания манхв в векторы размерности 384

2.  **Векторная база данных (ChromaDB):**
    *   Используется `ChromaDB` для хранения векторов и метаданных
    *   Реализован гибридный поиск: семантический поиск по векторам + фильтрация по метаданным (`where` clause в ChromaDB)
    *   Это позволяет отсеивать нерелевантные результаты (например, слишком старые или низкооцененные тайтлы) до этапа генерации

3.  **Генерация ответа (LLM):**
    *   Используется модель `Qwen2.5-0.5B-Instruct`
    *   Модель получает контекст (найденные манхвы) и запрос пользователя, после чего генерирует связный текст рекомендации на английском языке

4.  **Интерфейс (Gradio):**
    *   Веб-интерфейс для ввода запросов и настройки фильтров
    *   Визуализация результатов в виде таблицы и текстового блока с ответом
